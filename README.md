# Telugu-TranslatorTelugu Translator Extension

A simple browser extension that translates text into Telugu.

Features

Translate any text into Telugu.

Clean and easy-to-use popup interface.

Works directly inside your browser.

How to Install

Download or clone this repo:

git clone https://github.com/vishnuuuvrdn/Telugu-Translator-Extension.git


Open your browser’s Extensions page.

Turn on Developer Mode.

Click Load unpacked and select this folder.

How to Use

Click the extension icon.

Type or paste the text you want to translate.

Press Translate.

See the Telugu result instantly.

Folder Structure
manifest.json   → Extension setup
popup.html      → Popup layout
style.css       → Popup styling
script.js       → Translation logic
icon.png        → Extension icon
